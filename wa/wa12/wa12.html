Quote Generator

I altered this website to add a button to tell you more about the other famous quotes the author of the original quote has. To do this i made a subsequent call to the api using the information from the original call.

The script gets the author slug which is used to search quotes by author. I made a subsequent call with the slug and got the quote count from that call. From there i concatenated the string to include the count authors name and an explanatory sentence.
